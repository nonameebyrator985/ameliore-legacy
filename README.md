# Améliore Legacy

Bienvenue dans le projet **Améliore Legacy**!

Ce projet est un outil conçu pour aider les développeurs à gérer et à améliorer la qualité de leur code hérité. Grâce à une série d'analyses et de recommandations, il vise à rendre le code plus propre, plus maintenable et plus performant.

## Fonctionnalités
- Analyse statique du code
- Identification des mauvaises pratiques
- Suggestions de refactorisation
- Rapports de qualité du code
- Intégration avec des systèmes de gestion de projet

## Installation
1. Clonez ce dépôt :
   ```bash
   git clone https://github.com/votre-utilisateur/ameliore-legacy.git
   ```
2. Installez les dépendances :
   ```bash
   cd ameliore-legacy
   npm install
   ```
3. Exécutez l'outil :
   ```bash
   npm start
   ```

## Utilisation
Il suffit d'indiquer le chemin du dossier contenant le code hérité. L'outil générera un rapport détaillé et proposera des améliorations.
